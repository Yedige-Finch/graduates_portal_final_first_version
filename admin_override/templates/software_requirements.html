Software requirements

For full and correct work with the site, the following technical requirements must be observed:

- using one of the latest browsers listed: Microsoft Edge, Microsoft Internet Explorer, Mozilla Firefox, Google Chrome, Safari, Yandex.Browser, Opera;

- permission to use cookies;

- JavaScript performance.

To enable cookie:

When using Microsoft Internet Explorer:

- you need to open the item "Internet Options", menu "Tools" of the browser. Go to the "Privacy" tab, click the "Advanced" button. Check the presence of marks on the “Accept” points in the main and third-party cookies and the absence of a mark on the paragraph “Override automatic processing of cookies”

When using Mozilla Firefox

- in the top menu, select "Tools", in it - "Settings". In the "Privacy" section that opens, check the "Accept cookies from sites" option.

When using Opera Browser:

- you need to select the item "Quick Settings" in the menu item "Tools", activate the option "Enable cookies"

When using Google Chrome:

- click on the “Configure and manage Google Chrome” icon, click on the “Options” item, in the window that opens, go to the “Advanced” tab, in the “Personal data”, “Content settings” section, set the value “Allow saving local data (recommended)”.
